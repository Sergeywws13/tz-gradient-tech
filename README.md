🔗 Сервис сокращения ссылок (URL Shortener)
Простой и эффективный микросервис для сокращения длинных ссылок, написанный на FastAPI с использованием SQLAlchemy и PostgreSQL. Проект включает веб-интерфейс, REST API, автоматическую нормализацию URL, счётчик переходов и статистику.

✨ Функциональность
Сокращение ссылок: отправьте длинный URL и получите короткий идентификатор (6 символов).

Редирект: переход по короткой ссылке автоматически перенаправляет на исходный URL.

Счётчик переходов: каждый переход увеличивает счётчик visits в базе данных.

Статистика: по короткому идентификатору можно узнать количество переходов.

Нормализация URL: если ссылка передана без протокола (например, google.com), автоматически добавляется https://.

Простой веб-интерфейс: интуитивно понятная страница для создания ссылок и просмотра статистики.

Готов к контейнеризации: Dockerfile и docker-compose.yml включены.

🛠️ Технологии
Python 3.11+

FastAPI – современный веб-фреймворк

SQLAlchemy 2.0 (асинхронный) – ORM

PostgreSQL – база данных

Alembic – миграции (опционально)

Pydantic – валидация данных

Uvicorn – ASGI-сервер

Docker / Docker Compose – контейнеризация

pytest, httpx, aiosqlite – тестирование
